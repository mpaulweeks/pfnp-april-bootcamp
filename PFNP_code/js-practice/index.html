JavaScript practice

Before we can do complex stuff with JavaScript, we have to pay our dues and learn some basics. It will be a little boring at times just working with text in the console, but soon, you will be making totally interactive web pages. Wax on, wax off, and soon you will be kicking butt.

Change the value in the <script>; tag at the bottom to match the js file that you are editing. Refresh the page to re-load your code. Open the Chrome Dev Tools Console (Mac: cmd+shift+j, Windows: ctrl+shift+j) to see your console output.

Cheat sheet

Comments
// this is a comment. it will not do anything, just a note for yourself /** this is also a comment: don't forget to close it! */
Logging to the console
console.log(stuff) merely prints whatever 'stuff' is to the JavaScript console.
Declarations
Constants that can never change: const pi = 3.14; Variables can change. New style: let myAge = 33; Old style: var myAge = 33; Any name in JavaScript should follow lowerCamelCase, like eBay or iPhone. Each word after the first is capital, no spaces. Numbers are okay but not at the first character.
Operators
Plus, minus, divide, multiply: + - / * Equal to, == Equal to and of the same type (use this one): === Inequality != Strict Inequality (use this one) !== Less than, greater than, less than or equal, greater than or equal: < > <= >= Or: || And: &&
Data Types
Strings are in single or double quotes: let name = "Vincent"; Numbers are not in quotes, you can do math with them. let myAge = 33; let myAgeLaterThisYear = myAge + 1; Boolean is just true or false: const areBikesRad = true; const myAge = 19;
const canVote = myAge >= 18; Arrays are lists of things of any type let myFriends = ['Ben', 'Phillip', 'Paola']; Objects are key-value like structures like a dictionary: 
              
let hotSauce = {
  name: 'Peri peri',
  origin: 'South Africa',
  color: 'Orange',
  numBottles: 3
  ingredients: [
    'garlic',
    'salt',
    'basil',
    "bird's eye peppers",
    'vinegar',
    'cilantro',
    'olive oil',
    'paprika',
    'lemon'
    ]
};
Control flow
In an if/else statement, the first block with a true condition will run 
                
if (someCondition) {
  // do stuff if someCondition is true
} else if (someOtherCondition){
  // do stuff if someOtherCondition is true
} else {
  // if both above are false, this will run
}
Functions
A few ways to declare: 
              
function doSomeThings(){
  console.log('I am in a function.');
}
With a parameter 
              
function isVincentsEmail(email){
  return email === 'trivett@gmail.com';
}
A return value is the output that can be saved or passed to another function or whatever you want.
Invocation
After you declare a function, you must invoke it otherwise nothing happens. 
              
doSomeThings();

const checkWrongEmail = isVincentsEmail('trivett@hotmail.com'); // false
const checkRightEmail = isVincentsEmail('trivett@gmail.com'); // true
